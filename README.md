# 🚀 Tes Live Coding - Divisi Akademik

Selamat datang di tahap seleksi Live Coding! Tes ini dirancang untuk menguji logika, kemampuan pemecahan masalah, kreativitas ide, dan cara Anda mempresentasikan solusi teknis.

## 📋 Ketentuan Tes
Pada sesi ini, Anda dibebaskan untuk menentukan **ide atau tujuan aplikasi** yang akan dibuat (Contoh: Kasir, Manajemen Tugas, Kalkulator Kesehatan, dll). Namun, aplikasi wajib memenuhi kriteria di bawah ini:

### 1. Kriteria Ide & Problem Solving
* **Identifikasi Masalah:** Aplikasi harus mampu menyelesaikan satu masalah sederhana.
* **Kreativitas:** Ide aplikasi murni dari pemikiran peserta.
* **Penjelasan:** Peserta mampu menjelaskan alur logika aplikasi saat sesi presentasi.

### 2. Persyaratan Teknis (Wajib Ada)
Aplikasi berbasis **Console** yang dibangun menggunakan bahasa **Go (Golang)** dengan komponen:
* **Variables:** Penggunaan tipe data yang tepat (String, Int, Float).
* **Function:** Minimal terdapat satu fungsi untuk memproses data.
* **Looping:** Proses yang berjalan berulang hingga kondisi tertentu terpenuhi.
* **Logic Flow (If-Else):** Pengambilan keputusan berdasarkan input pengguna.
* **Input-Proses-Output:** Alur data yang jelas dari awal hingga akhir.
