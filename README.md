## 📝 Studi Kasus: "Digital Saving Tracker"
Anda diminta membuat aplikasi konsol yang membantu pengguna melacak target tabungan mereka.

### Kriteria Wajib:
1. **Input:** Nama, Target Uang, dan Setoran harian.
2. **Proses:** - Gunakan `Function` untuk menghitung sisa target.
   - Gunakan `Loop` agar pengguna bisa setor berkali-kali.
   - Gunakan `If-Else` untuk menentukan apakah target sudah tercapai.
3. **Output:** Menampilkan status tabungan dan pesan keberhasilan.

## 🛠️ Persyaratan Teknis
- Bahasa Pemrograman: **Go (Golang)**
- Konsep: Variable, Arithmetic, Logic Flow, Loop, Function.
